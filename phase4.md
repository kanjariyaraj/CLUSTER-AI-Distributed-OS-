# Phase 4: Cluster System & Distributed Computing

## Goal
Enable multiple AIDOS nodes to work together as a distributed AI cluster.

## Tasks
- [ ] **Task 4.1: Node-to-Node Communication**
  - Implement a basic TCP socket protocol for task sharing.
- [ ] **Task 4.2: Node Server Development**
  - Create `node_server` to listen for tasks from the controller.
- [ ] **Task 4.3: Controller Implementation**
  - Create `controller` to split and distribute AI workloads.
- [ ] **Task 4.4: Cluster Orchestration**
  - Add discovery mechanism or static configuration for cluster members.

## Deliverables
- `node_server` and `controller` binaries in `/opt/aidos/`.
- Working cluster demonstration on local network.
